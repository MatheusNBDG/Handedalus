## Versão beta do aplicativo Handedalus

Está é a versão beta 1.1 de um aplicativo android direcionado para o gerenciamento de livros das bibliotecas da USP (Universidade de São Paulo) a ser desenvolvido em prol da coveniência do autor (cançei de esqueçer de renovar essas coisas já...).

Suas principais funcionabilidades são:

- "One Login" : Quando a caixa "Salvar usuario e senha" for marcada não será necessário transpassar pela tela de login para entrar no aplicativo, ou seja, você será redirecionado diretamente a lista dos seus livros.

- "Lista de Livros" : Exibe as principais informações sobre os livros pegado emprestado na biblioteca (beta: maximo 10 livros) em ordem crescente de dias para o vencimento.

- "Renovar selecionados" : Renova os livros selecionados.

- "Avisar-me em n dias" : Configura uma notificação para te avisar sobre o vencimento do livro mais proximo para o numero de dias configurado.

- "In The Shadows" : Aproximadamente uma vez ao dia o aplicativo procurará sozinho, mesmo estando fechado, se tem um livro vencendo em três dias e te avisará sobre. Só funciona se a opção One Login for ativada!


Além disso, é interessante explicitar algumas considerações sobre a atual versão para caso você queira trabalhar nela:

- Não compreendo o atual layout como o da versão final. Predento confifugar um layout mais fino e minimalista para a versão final.

- Há um bug com "Avisar-me em n dias", notificações futuras não gravam em quantos dias o livro está para vençer e mostra null no lugar. É facil de se resolver.

## Como instalar 

Basta baixar o apk disponível acima para um dispositivo android e instala-lo (Pokemon go feelings...).
