## Versão beta do aplicativo Handedalus

Está é a versão beta 1.0 de um aplicativo android direcionado para o gerenciamento de livros das bibliotecas da USP (Universidade de São Paulo) a ser desenvolvido em prol da coveniência do autor (cançei de esqueçer de renovar essas coisas já...).

É interessante explicitar algumas considerações sobre a atual versão para caso você queira trabalhar nela:

- Não compreendo o atual layout como o da versão final. Predento confifugar um layout mais fino e minimalista para a versão final.

- Tecnicamente, o programa foi programado para monitorar os seus livros enquando o aplicativo estiver fechado. Mas, esta função não está funcionando corretamente no momento.
